## 部署方式
- 后端运行在树莓派 5（`rpi5.lan`，IP: `192.168.6.220`），端口 8000
- 前端生产部署：nginx (80) → 代理 `/api`、`/ws` 到后端 8000
- 前端开发：`npm run dev` 在 Mac `127.0.0.1:8080`，Vite proxy 转发 `/api`、`/ws` 到 `rpi5.lan:8000`
- 后端需要在树莓派上编译部署（因为 gopacket/pcap 依赖 CGO）
- **重要**：`.env.production` 不应设置 `VITE_API_URL`，让前端走同源 nginx 代理
